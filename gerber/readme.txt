This folder contains the Gerber files needed to make the PC board.
File dspsdr40_rb26.zip is for my 16-Bit board with the 26-pin header.
File dspsdr40_db25.zip is for the board with the DB25 connector.
Other than the input connector the boards are identical.
The PCB is for the Analog Devices LTC1605CSW#PBF / LTC1605ISW#PBF ADC chip with the 28-SOIC (0.295"", 7.50mm Width) package.
